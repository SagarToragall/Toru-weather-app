# TORU Weather App

I built the project using HTML, Tailwind CSS, and JavaScript for the frontend, Node.js and Express.js for the backend, integrated OpenWeather API for real-time data, used lowdb for storage, Chart.js for visualization, and deployed it using GitHub and Render

## Prerequisites

- Node.js (LTS recommended)
- npm

## Setup

```bash
npm install
```

## Run

```bash
node server.js
```

## Notes

- `node_modules/` is intentionally ignored via `.gitignore`.
